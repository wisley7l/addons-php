Countdown Plugin

API

  • width
  • height
  • fillWidth
  • color
  • gradient
  • gradientColors
  • targetDate
  • gradientColors
  • outline
  • outlineColor
  • days
  • hours
  • minutes
  • seconds
  • daysText
  • hoursText
  • minutesText
  • secondsText

width: the width of each circle. Default: 0

height: the height of each circle. Default: 0

fillWidth: the width of the border of each circle. Default: 0

color: the color of the border of each circle. Default: ''

gradient: boolean value indicating the use of gradient. Default: false

gradientColors: array containing the first and last gradient colors (Example: ['#26cfb1','#6c4788']). Default: []

outline: boolean value indicating the use of an outline. Default: false

outlineColor: string indicating the outline color. Default: '#eff0f4'

targetDate: the target date of the countdown, it must be set using the javascript Date Object (Example: new Date(2015, 1, 27)). Default: false

days: boolean indicating display of days circle. Default: true

hours: boolean indicating display of hours circle. Default: true

minutes: boolean indicating display of minutes circle. Default: true

seconds: boolean indicating display of seconds circle. Default: true

daysText: string indicating text to display for days circle. Default: 'days'

hoursText: string indicating text to display for hours circle. Default: 'hours'

minutesText: string indicating text to display for minutes circle. Default: 'minutes'

secondsText: string indicating text to display for seconds circle. Default: 'seconds'

Accordion Plugin

API

  • autoClose
  • startOpen
  • easing
  • speed
  • mainClass
  • itemClass
  • contentClass
  • selectedClass

autoClose: boolean value, if true, when opening an item the others close automatically. Default: true.

startOpen: index of item to be opened initially. Default: 0

easing: easing applied when opening items, compatible with any jQuery easing. Default: 'swing'

speed: the speed of the animation. Default: 600

mainClass: string indicating the class of the accordion container. Default: '.accordion'

itemClass: string indicating the class of the element that is clicked to show the content. Default: '.accordion-item'

mainClass: string indicating the class of the container of the content to be displayed. Default: '.accordion-item-content'

mainClass: string indicating the class to be applied to the element referenced by the itemClass when the element is open. Default: 'selected'

Alerts Plugin

API

  • template
  • title
  • paragraph
  • timestamp
  • imgSrc
  • iconClass
  • buttonSrc
  • buttonText
  • x
  • y
  • xOffset
  • yOffset
  • alertSpacing
  • autoClose
  • lifetime
  • fade
  • fadeDelay

template: template to be used for the alert. ('messageInfo', 'messageSuccess', 'messageError', 'item', 'review', 'survey') Default: "messageInfo".

title: title for the alert. Default: "Title".

paragraph: paragraph for the alert. Default: "Lorem ipsum dolor sit amet.".

timestamp: timestamp for the alert. Default: "X hours Ago".

imgSrc: source of the image for the alert. Default: "images/avatars/avatar_01.jpg".

iconClass: class of the simplelineicon icon to be displayed. Default: "icon-heart".

buttonSrc: links for the buttons, comma separated, served in order. Default: ['#', '#'].

buttonText: text for the button. Default: "".

x: horizontal direction of the alert. ("right", "left") Default: "right".

y: vertical direction of the alert. ("top", "bottom") Default: "top".

xOffset: separation between the alert and the horizontal margin. Default: 40.

yOffset: separation between the alert and the vertical margin. Default: 40.

alertSpacing: separation between stacked alerts. Default: 20.

autoClose: if true, closes the alert after x miliseconds ( uses lifetime setting ). Default: true.

lifetime: sets the time ( in miliseconds ) the alert stays before hiding. Default: 6000.

fade: if true, fade effect is applied to all transitions. Default: true.

fadeDelay: delay of the fade effect. Default: .3.

Setting up a popup

MagnificPopup

  • Add the MagnificPopup CSS and javascript dependencies

  • Copy the popup template that you want to use, paste it inside the body tag and add it "mfp-hide" class to hide it

  • Assign it an ID and add it to the href attribute of the button that will open it, also add a class to identify the button, i.e:
    <a href="#yourid" class="open-login-popup">Login</a>

  • Finally add this for MagnificPopup to work when you click that button:

    $('.open-login-popup').magnificPopup({
    type: 'inline',
    closeMarkup: '<div class="close-btn mfp-close"><svg class="svg-plus"><use xlink:href="#svg-plus"></use></svg></div>'
    });
